Résumé du jeu :

Genre du Jeu :

Aventure-action et Infiltration

Le but du jeu :

Vous jouez Adien Pearce un hacker qui utilise le ctOS (système informatique qui gère toutes les infrastructures connectés) pour parvenir à ses fins.
Ce jeu est un GTA-like qui a pour thème principal le hacking, qui est un thème peu exploité dans le milieu du jeux vidéos.

Avis :

Yannis :

Ce jeu a reçu pas mal de critique assez négatives qui ne sont pour moi pas mérités.
En effet j'ai adoré joué ce jeu, l'histoire est bien faite, vous pouvez tout hacker, les phases de hackage sont assez réalistes et pas mal de spécificté de l'informatique/hackage sont abordés.
Je donne la note de 9/10 à ce jeu.
